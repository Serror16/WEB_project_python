### На данный момент реализовано:

* Все настройки парсеров и рендереров перенесены в `settings.py` (`REST_FRAMEWORK`).

* Добавлена поддержка форматов `JSON` и `XML` для всех эндпоинтов.

* Сделал поддержку суффиксов `.json` и `.xml` в конце URL, для тестирования без доп танцев с бубном

* Написал регулярное выражение для `tax-status`, чтобы расширение(суффикс) не попадал в имя переменной.

* Методы get и post принимает аргумент format, что позволяет серверу автоматически выбирать тип ответа на основе суффикса или заголовка `Accept`.

### Итог:
Сервер корректно принимает `XML`/`JSON`, и отдает результат в формате(`XML`/`JSON`) взависимости от страны.
